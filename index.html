export default {
  async fetch(request) {
    if (request.method === 'OPTIONS') {
      return new Response(null, {
        headers: {
          'Access-Control-Allow-Origin': '*',
          'Access-Control-Allow-Methods': 'GET, OPTIONS',
        }
      });
    }
    const url = new URL(request.url);
    const target = url.searchParams.get('url');
    if (!target) return new Response('no url', { status: 400 });
    const allowed = ['query1.finance.yahoo.com','query2.finance.yahoo.com','stooq.com','finviz.com'];
    const targetUrl = new URL(target);
    if (!allowed.some(d => targetUrl.hostname.includes(d))) {
      return new Response('not allowed', { status: 403 });
    }
    const resp = await fetch(target, {
      headers: { 'User-Agent': 'Mozilla/5.0' }
    });
    const body = await resp.text();
    return new Response(body, {
      status: resp.status,
      headers: {
        'Content-Type': resp.headers.get('content-type') || 'application/json',
        'Access-Control-Allow-Origin': '*',
        'Cache-Control': 'public, max-age=300',
      }
    });
  }

Alles over VSA trading voor Nederlandse beleggers

Wat is Volume Spread Analysis (VSA)? +

VSA analyseert de relatie tussen volume, prijsbereik en sluitingsprijs. Ontwikkeld door Tom Williams op basis van Richard Wyckoff. Het helpt je zien wat professionele traders "smart money" doen, nog voordat de koers beweegt.

Hoe gebruik ik de VSA Trader tool? +

Typ een aandelensymbool bovenaan zoals AAPL voor Apple of ASML.AS voor ASML en klik op ANALYSEER. Je krijgt direct een volledig rapport: signalen, bias, trendsterkte, candlestickgrafiek en checklist. Geen account of betaling nodig.

Welke aandelen kan ik analyseren? +

Amerikaanse aandelen AAPL, TSLA, NVDA, AMD, PLTR, SOFI werken direct. AEX aandelen gebruik je met .AS extensie: ASML.AS, PHIA.AS, HEIA.AS. Ook ETFs en crypto-stocks zijn beschikbaar via Yahoo Finance.

Wat betekenen VWAP, EMA en ATR? +

VWAP is de gemiddelde prijs gewogen naar volume. Boven VWAP = bullish, eronder = bearish.
EMA-20 / EMA-50 is het exponentieel voortschrijdend gemiddelde. EMA-20 boven EMA-50 = bullish trend.
ATR meet de gemiddelde dagelijkse beweging. Gebruik 1,5× ATR als stop-loss afstand.

Is de tool geschikt voor beginners? +

Ja. Elk signaal heeft een uitlegknop. De Leerbibliotheek legt alle concepten stap voor stap uit. De interactieve quiz helpt je kennis te testen. Geen voorkennis vereist.
