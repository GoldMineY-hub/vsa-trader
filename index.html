Proxy Instellingen
Data ophalen via allorigins proxy · stel eigen Cloudflare Worker in voor snellere verbinding

Yahoo Finance blokkeert directe verzoeken vanuit de browser. De app gebruikt allorigins.win als doorstuurproxy · dit werkt maar is soms traag (10–30 seconden).
Voor de snelste verbinding: zet je eigen gratis Cloudflare Worker op. Duurt 5 minuten, geen creditcard nodig.

📋 Stap-voor-stap instructie
Stap 1 · Ga naar workers.cloudflare.com → gratis account
Stap 2 · Klik Create Worker → Start with Hello World! → Deploy
Stap 3 · Selecteer alle code (Ctrl+A), verwijder het
Stap 4 · Kopieer de Worker-code rechts en plak hem in
Stap 5 · Klik Deploy → kopieer je Worker URL
Stap 6 · Plak de URL hieronder en klik Opslaan
Worker code:
export default {
  async fetch(request) {
    if (request.method === 'OPTIONS') {
      return new Response(null, {
        headers: {
          'Access-Control-Allow-Origin': '*',
          'Access-Control-Allow-Methods': 'GET, OPTIONS',
        }
      });
    }
    const url = new URL(request.url);
    const target = url.searchParams.get('url');
    if (!target) return new Response('no url', { status: 400 });
    const allowed = ['query1.finance.yahoo.com','query2.finance.yahoo.com','stooq.com','finviz.com'];
    const targetUrl = new URL(target);
    if (!allowed.some(d => targetUrl.hostname.includes(d))) {
      return new Response('not allowed', { status: 403 });
    }
    const resp = await fetch(target, {
      headers: { 'User-Agent': 'Mozilla/5.0' }
    });
    const body = await resp.text();
    return new Response(body, {
      status: resp.status,
      headers: {
        'Content-Type': resp.headers.get('content-type') || 'application/json',
        'Access-Control-Allow-Origin': '*',
        'Cache-Control': 'public, max-age=300',
      }
    });
  }
};
