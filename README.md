# VSA Trader

A free Volume Spread Analysis tool built for European day traders. 
VSA helps you read institutional activity in the market by analysing 
the relationship between price spread, volume and closing position.

## What is VSA?

Volume Spread Analysis was developed by Tom Williams, based on the 
work of Richard Wyckoff. It identifies when institutional traders 
(banks, hedge funds) are accumulating or distributing positions — 
before the move happens.

## Features

- Real-time VSA signal detection
- Works on US markets from Europe
- No subscription, no registration required
- Powered by Cloudflare Workers + Finnhub API
